# AIMicroAgent.com Static Site

Marketing site for AIMicroAgent.com, positioning AI micro-agents as composable microservices.

## Structure
- `index.html` - Home
- `approach.html` - Philosophy and doctrine
- `architecture.html` - Reference architecture
- `catalog.html` - Agent catalog
- `fine-vs-coarse.html` - Fine vs coarse breakdown
- `docs.html` - Getting started docs
- `assets/` - CSS, JS, and SVG assets

## Run locally
Open `index.html` in a browser or serve the folder with a simple static server:

```bash
python3 -m http.server 8080
```

Then visit `http://localhost:8080`.
